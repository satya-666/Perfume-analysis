# Project Overview

**Tech Stack:** React.js, Firebase, JavaScript

A multi‑feature web platform that delivers a rich content experience:

- **News Hub:** Aggregates and displays over 20 news categories, enabling users to stay informed across topics.
- **Blogging Suite:** Full‑featured blogging tools for creating, editing, and publishing posts.
- **Weather Widget:** Real‑time weather updates integrated directly into the dashboard.
- **Authentication:** Secure Google and Email sign‑in via Firebase Authentication.
- **Personalization:** Bookmarking capability and user‑specific content management.
- **Responsive Dashboard:** An adaptive, mobile‑first UI that adjusts seamlessly to any screen size, providing a smooth and engaging user experience.

The application leverages Firebase for real‑time database, hosting, and auth, while React.js delivers a performant, component‑based front‑end.
